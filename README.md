# 💰 Finance Analyzer (Budget vs Actual)

## 📊 Overview

This project analyzes personal expenses and compares them with planned budget using Python.

## ⚙️ Features

* Read Excel data using pandas
* Category-wise expense calculation
* Budget vs Actual comparison
* Difference (Overspending detection)
* Bar chart visualization

## 📁 Dataset

* Transactions sheet → actual expenses
* Budget sheet → planned budget

## 📊 Output

* Category-wise comparison
* Difference column (Budget - Actual)
* Graph showing spending behavior

## 🛠️ Tech Used

* Python
* pandas
* matplotlib

## 🚀 How to Run

```bash
pip install -r requirements.txt
python main.py
```
